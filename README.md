# Portfólio Profissional | Renzo Fernandes

Aplicação web desenvolvida para apresentar meu perfil profissional, competências técnicas, projetos, formação acadêmica e canais de contato. O objetivo deste portfólio é reunir, em uma experiência visual moderna e responsiva, informações relevantes para recrutadores, profissionais de tecnologia e empresas interessadas em conhecer minha trajetória e meus projetos.

## Sobre o projeto

Este portfólio foi construído com foco em clareza, performance, responsividade e apresentação profissional. A interface utiliza uma estética tecnológica em tema escuro, com animações suaves, seções bem definidas e navegação por âncoras para facilitar a leitura do conteúdo.

A aplicação apresenta:

- Resumo profissional e objetivos de carreira.
- Seção sobre formação, interesses e perfil técnico.
- Lista organizada de habilidades por categoria.
- Carrossel responsivo de projetos.
- Formação acadêmica, certificações e idiomas.
- Links diretos para e-mail, GitHub, LinkedIn e WhatsApp.

## Objetivo

O projeto foi desenvolvido como uma vitrine profissional para apoiar minha busca por oportunidades na área de tecnologia, especialmente em desenvolvimento de software, dados e sistemas corporativos.

Além de apresentar informações sobre meu perfil, o repositório também demonstra práticas de desenvolvimento front-end com React, TypeScript, componentização, responsividade, estilização com Tailwind CSS e build estático com Vite.

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Vite React SSG
- Tailwind CSS
- React Icons
- ESLint

## Principais recursos

- Layout responsivo para desktop, tablet e mobile.
- Navegação suave entre seções.
- Header fixo com menu desktop e mobile.
- Animações de entrada ao rolar a página.
- Background visual com canvas e efeitos sutis.
- Carrossel de projetos com navegação por botões, paginação e suporte a arraste.
- Estrutura preparada para publicação estática.
- Organização de conteúdo com foco em leitura rápida por recrutadores.

## Como executar localmente

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Acesse a pasta do projeto:

```bash
cd my-portfolio
```

Instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Depois, acesse o endereço exibido no terminal, normalmente:

```text
http://localhost:5173
```

## Scripts disponíveis

```bash
npm run dev
```

Executa a aplicação em modo de desenvolvimento.

```bash
npm run build
```

Gera a versão final estática da aplicação.

```bash
npm run preview
```

Executa uma prévia local da versão de produção.

```bash
npm run lint
```

Executa a análise de lint nos arquivos do projeto.

## Build e qualidade

O projeto utiliza Vite React SSG para gerar uma versão estática otimizada, adequada para deploy em plataformas como Vercel, Netlify ou GitHub Pages.

Antes da publicação, recomenda-se executar:

```bash
npm run lint
npm run build
```
